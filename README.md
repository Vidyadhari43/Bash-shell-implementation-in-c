# Basic Shell Implementation in c
### Features:
1) Indicates current working directory.
2) User can specify commands either by realtive or absolute pathnames.
3) Gives output for all commands for which an executable file located in one of the directories listed in the PATH environment variable.
4) Gives recent 100 commands for history command.
5) Able to redirect STDIN and STDOUT for the new process by using < and >.
